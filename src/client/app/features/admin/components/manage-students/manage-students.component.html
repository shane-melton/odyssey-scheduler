Bulk Operations

This section will contain settings for importing/updating batches of students from CSV's and clearing all students.


Import New Students

Import a fresh set of students into the database and remove all existing students from the system. This cannot be undone!

Update Students

Import a subset of students which will update any existing students with the same student number. If a student number is not found a new student will be created. This cannot be undone!

Clear All Students

Remove all student data from the system. This will also remove any makeup information and cannot be undone!
